# MGS.Finder

## Summary
- Unity plugin to control Unity Play/Pause/Step, open unity path and scene.


## Ability

- Quick control Unity Play/Pause/Step, open unity path and scene.

## Usage
1. Find the menu item "Tool/Assistant/Finder" in Unity editor menu bar and click it or press the Alt+F shortcut key to open the finder window.
1. Use the quick buttons to operate Unity.

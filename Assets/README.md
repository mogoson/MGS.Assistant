# MGS.Assistant

## Summary

- Assistant of Unity for developer.

## Ability

- [Assemblyman](.\Editor\Assemblyman\README.md) to to analysis assembly references.

- [AssetFilter](.\Editor\AssetFilter\README.md) to check specification asset name.
- [AssetGuider](.\Editor\AssetGuider\README.md) to open Unity Asset use the OS default application.
- [CubemapRenderer](.\Editor\CubemapRenderer\README.md) to render a scene into a static Cubemap asset for Skybox material or reflective material.
- [Finder](.\Editor\Finder\README.md) to control Unity Play/Pause/Step, open unity path and scene.
- [Inspector](.\Editor\Inspector\README.md) to invoke method of MonoBehaviour script by custom button from Inspector tab.
- [MeshCombiner](.\Editor\MeshCombiner\README.md) to combin meshes to a mesh.
- [ScriptDiemaker](.\Editor\ScriptDiemaker\README.md) to insert header comments into the new script and mark company, copyright year, author name and create date.
- [TerrainModifier](.\Editor\TerrainModifier\README.md) to modify TerrainData heightmap.

## Install

- Unity --> Window --> Package Manager --> "+" --> Add package from git URL...

  ```text
  https://github.com/mogoson/MGS.Assistant.git?path=/Assets
  ```

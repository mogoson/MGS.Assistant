# MGS.assistant

## Summary

- Assistant of Unity for developer.

## Ability

- Finder to control Unity Play/Pause/Step, open unity path and scene.

## Install

- Unity --> Window --> Package Manager --> "+" --> Add package from git URL...

  ```text
  https://github.com/mogoson/MGS.Assistant.git?path=/Assets
  ```

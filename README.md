# MGS.Assistant

## Summary

- Assistant of Unity for developer.

## Ability

- Assemblyman to to analysis assembly references.
- AssetFilter to check specification asset name.
- AssetGuider to open Unity Asset use the OS default application.
- CubemapRenderer to render a scene into a static Cubemap asset for Skybox material or reflective material.
- Finder to control Unity Play/Pause/Step, open unity path and scene.
- Inspector to invoke method of MonoBehaviour script by custom button from Inspector tab.
- MeshCombiner to combin meshes to a mesh.
- ScriptDiemaker to insert header comments into the new script and mark company, copyright year, author name and create date.
- TerrainModifier to modify TerrainData heightmap.

## Install

- Unity --> Window --> Package Manager --> "+" --> Add package from git URL...

  ```text
  https://github.com/mogoson/MGS.Assistant.git?path=/Assets
  ```
